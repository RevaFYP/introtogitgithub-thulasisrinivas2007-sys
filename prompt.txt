Write a complete C program for a console-based 2D Graphics Editor using only standard C libraries (stdio.h, stdlib.h, math.h).

Requirements:

1. Create a 60x60 drawing canvas using a 2D character array.

2. Initialize the canvas with blank characters represented by '_'.

3. Define a structure named Object with:

   * int id
   * int type (1=Line, 2=Rectangle, 3=Triangle, 4=Circle)
   * Coordinates:

     * x1, y1
     * x2, y2
     * x3, y3
   * radius

4. Maintain an array of up to 100 objects and keep track of the number of objects.

5. Implement the following functions:

   * initializeCanvas()
   * displayCanvas()
   * plot()
   * drawLine() using Bresenham's Line Algorithm
   * drawRectangle()
   * drawTriangle()
   * drawCircle() using Midpoint Circle Algorithm
   * drawObject()
   * addObject()
   * deleteObject()
   * modifyObject()

6. Menu-driven program:

   1. Add Object
   2. Delete Object
   3. Modify Object
   4. Display Picture
   5. Exit

7. When adding an object:

   * Assign a unique ID automatically.
   * Ask the user for coordinates.
   * Store the object in the array.
   * Draw it on the canvas using '*' character.

8. When deleting an object:

   * Ask for Object ID.
   * Remove it from the array.
   * Erase it from the canvas.

9. When modifying an object:

   * Ask for Object ID.
   * Remove the old drawing.
   * Take new details from the user.
   * Redraw the updated object.

10. Display the canvas in a readable format where blank cells appear as spaces and drawn pixels appear as '*'.

11. Use switch-case statements wherever appropriate and keep the code modular with separate functions.

12. Include sufficient comments explaining every major section.

Generate the complete compilable C source code.
